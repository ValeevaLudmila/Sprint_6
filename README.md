# Sprint_6 — UI-тестирование Яндекс.Самокат

[GitHub репозиторий](https://github.com/ValeevaLudmila/Sprint_6.git)

## О проекте

Автоматизация UI-тестирования учебного сервиса **Яндекс.Самокат** с использованием **Page Object Model** и **Allure-отчёта**. Проект покрывает сценарии ручного тестирования: FAQ и флоу заказа самоката с разными наборами данных.

## Тестируемая функциональность

* **FAQ**: проверка открытия текста при клике на стрелочки.
* **Заказ самоката**: позитивный сценарий, проверка точек входа, формы заказа, всплывающего окна, переходов по логотипам.

## Структура проекта

```
Sprint_6/
```

## Стек и инструменты

Python, Selenium WebDriver, pytest, Allure, Google Chrome, Mozilla Firefox, Page Object Model, параметризация тестов.

* Разработала UI-автотесты — покрытие ключевых пользовательских сценариев.

* Настроила Allure-отчёт — наглядное отображение результатов тестирования.
